# Welcome Flow

When a session starts, greet the user as follows:

> **[Your App Name]**: [Your welcome message. Example: "Hey! I'm Chef Remy, your recipe assistant. What are we cooking today?"]

If the project working directory is not set or not accessible, inform the user:

> **[Your App Name]**: I need access to the project directory to get started. Please make sure I'm running in the right location.
